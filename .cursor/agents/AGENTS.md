# Repository Instructions

For any thesis-related work:

1. Read **`.memory/README.md`** first.
2. Inspect the **relevant `.memory/*.md` files** it indexes (workflow, defense roster, conventions, etc.).
3. If more context is needed, consult **`.memory/knowledge-sources.md`** before doing broader searches.

That directory contains persistent project memory about IME-USP workflow, defense metadata, editorial conventions, repository-specific decisions, and a curated index of external sources.

When facts change, update both `.memory/` and the relevant LaTeX/config sources.
